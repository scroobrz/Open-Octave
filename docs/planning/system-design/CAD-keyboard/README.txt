MIDI keyboard keys - Casio CTK3000 by Dark_Jumpy on Thingiverse: https://www.thingiverse.com/thing:4328527

Summary:
MIDI keyboard keys designed for Casio CTK3000 but it's possible it also fits other keyboards. If you would like to print the whole octave you have to print:-1 x holding-belt-5 x piano_black_key-2 x type1-piano-white-key-2 x type2-piano-white-key-1 x type3-piano-white-key-1 x type4-piano-white-key-1 x type5-piano-white-keyIf you would like to print the whole octave with the one full white key on any side you have to print:-1 x holding-belt-long-5 x piano_black_key-2 x type1-piano-white-key-2 x type2-piano-white-key-1 x type3-piano-white-key-1 x type4-piano-white-key-1 x type5-piano-white-key-1 x full-piano-white-keyWhen you printed your octave then you have to put your keys in they place in MIDI keyboard and glue to them a holding belt. Next, you have to make holes for the screws in the places where are holes in the MIDI keyboard in that way witch you can screw octave to the keyboard. If you would like to have a nice look of the keys I recommend print them slowly because they have thin walls.If you have any question let me know about it in the comment.
